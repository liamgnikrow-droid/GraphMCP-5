# Только YAML ссылки

> [!abstract] Requirement Context
> **ID:** `REQ-YAML_LINKS_ONLY` | **Status:** `Approved`

## Description
Все связи между узлами в Obsidian должны быть строго в формате YAML фронтматера. Использование [[вики-ссылок]] внутри текста запрещено.
