# System: Evolution Protocol (Meta-Graph Changes)

> [!abstract] Requirement Context
> **ID:** `REQUIREMENT-SYSTEM__EVOLUTION_PROTOCOL__META_GRAPH_C` | **Status:** `Draft`

## Description
**Spec Ref:** `[6.3]`

Протокол внесения изменений в Мета-Граф:
- Агент имеет право только предлагать изменения через `propose_change`.
- Прямая модификация Kernel Space разрешена только в специальном Genesis Mode (Host Only).
- Все изменения Мета-Графа должны быть одобрены внешним Архитектором (Архитектор Мета-Законов).
