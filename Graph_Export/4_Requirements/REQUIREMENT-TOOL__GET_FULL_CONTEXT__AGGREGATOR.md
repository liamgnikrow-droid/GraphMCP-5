# Tool: Get Full Context (Aggregator)

> [!abstract] Requirement Context
> **ID:** `REQUIREMENT-TOOL__GET_FULL_CONTEXT__AGGREGATOR` | **Status:** `Draft`

## Description
Агрегатор полного контекста для задачи (Context Aggregation).
Функциональные требования:
- Сбор всех релевантных данных в один ответ.
- Поиск семантически близких узлов.
- Извлечение связанных спецификаций и требований по связям графа.
- Отображение активных системных ограничений (Constraints).
- Анализ влияния (Impact Preview) планируемых действий.
