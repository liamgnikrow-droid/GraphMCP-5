# Tool: Illuminate Path (Vertical Trace)

> [!abstract] Requirement Context
> **ID:** `REQUIREMENT-TOOL__ILLUMINATE_PATH__VERTICAL_TRACE` | **Status:** `Draft`

## Description
Подсветка логического пути в графе (Traceability).
Функциональные требования:
- Поиск точки входа (Entry Point) по семантическому запросу.
- Трассировка пути вверх к корню (Idea) и вниз к листьям (Task).
- Извлечение контента всех узлов на найденном пути.
- Отображение боковых связей (Lateral connections).
