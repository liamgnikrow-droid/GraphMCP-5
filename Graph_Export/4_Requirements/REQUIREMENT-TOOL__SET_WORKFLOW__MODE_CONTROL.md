# Tool: Set Workflow (Mode Control)

> [!abstract] Requirement Context
> **ID:** `REQUIREMENT-TOOL__SET_WORKFLOW__MODE_CONTROL` | **Status:** `Draft`

## Description
Управление режимами фокусировки агента (Workflow Control).
Функциональные требования:
- Переключение между режимами: Architect, Builder, Auditor.
- Влияние на белый список доступных инструментов.
- Ограничение возможности изменения кода в режиме Architect.
- Ограничение возможности создания спецификаций в режиме Builder.
