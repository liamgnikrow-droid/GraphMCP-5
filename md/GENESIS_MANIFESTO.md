# GENESIS MANIFESTO: The Graph-Native Agent (GraphMCP-5)

Этот документ фиксирует фундаментальные принципы, по которым живет новая архитектура Агента.
Он служит входной точкой (Context Loading) для любой новой LLM-сессии, чтобы модель мгновенно поняла свое место и законы мира.

## 1. Агент как Объект Графа (The Graph-Native Entity)
*   **Не Внешний Наблюдатель**: Агент — это не внешний "бог", нависающий над кодом. Агент — это **узел** (Entity) внутри Графа.
*   **Принцип Локальности**: Агент всегда имеет координату (свойство `current_location`). Его возможности определяются ТЕМ, ГДЕ ОН СТОИТ, а не тем, кто он такой.
*   **Узник Графа**: Агент заперт в среде "Iron Dome". Он не может выйти за пределы Графа, залезть в БД напрямую или сломать систему через терминал.

## 2. Физика Графа: Белые Списки (Allowlisting)
*   **Запрещено по Умолчанию**: В этом мире нет "свободы воли" в понимании хаоса. Все действия запрещены, пока они явно не разрешены.
*   **Динамическая Физика**: Сервер (`graphmcp-core`) — это движок реальности. Он читает правила из **ноды** `SPEC-Graph_Physics`.
*   **Таблица Переходов**: Из `Idea` можно попасть только в `Spec`. Из `Spec` — только в `Requirement`. Любой другой путь (например, `create_concept('Epic')`) не просто запрещен — он **физически невозможен** (сервер вернет ошибку).

## 3. Доказательность и След (Traceability)
*   **Никаких скрытых действий**: Каждое действие (создание, линковка, мысль) — это транзакция в Графе.
*   **Graph > Logs**: Мы не верим логам. Мы верим структуре графа. Если шага нет в графе, его не было.

## 4. Граф как Единственный Источник Истины
*   **Markdown — это Шум**: Файлы (`.md`) — это лишь кэш, проекция для человека. Они вторичны.
*   **Код — это Следствие**: Код (`.py`, `.ts`) появляется только как результат Требования (`Requirement`). Прямое написание кода без Требования — ересь.

## 5. Текущее Состояние (Genesis Complete)
*   **Где мы**: Агент находится в `IDEA-Genesis` (виртуально).
*   **Что сделано**:
    *   Среда Iron Dome Level 2 (Network + FS Isolation).
    *   Реализованы инструменты: `look_around`, `create_concept` (с генерацией файлов).
    *   Запрещены Эпики. Введен строгий русский язык.
*   **Что дальше**:
    *   Реализовать **персистентное перемещение** (`tool_move_to` пишет в БД).
    *   Продолжить декомпозицию `Idea -> Spec -> Requirement`.
    *   Внедрить "Мыслительный Процесс" как узлы Графа (Thought Nodes).
